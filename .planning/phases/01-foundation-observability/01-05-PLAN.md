|---|---|

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-12 | Information Disclosure | evento Sentry | mitigate | sendDefaultPii false, hooks allowlist e testes canário. |
| T-01-13 | Denial of Service | captura excessiva | mitigate | Warn esperado não captura; traces desabilitados nesta fase. |
| T-01-14 | Repudiation | duplicação de evento | mitigate | Captura única com correlation ID e delegação padrão. |
| T-01-SC | Tampering | npm install | mitigate | Checkpoint humano bloqueante para @sentry/node 10.59.0. |
