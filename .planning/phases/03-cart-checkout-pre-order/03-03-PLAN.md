## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| cliente -> validadores de checkout | Cliente envia email, endereco, CEP, UF e CPF/CNPJ nao confiaveis. |
| validadores -> storage de cart/address | PII normalizada entra em storage server-side. |
| validadores -> logs/Sentry/respostas | Erros podem vazar PII se nao forem saneados. |
