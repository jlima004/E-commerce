- Mandatory Gelato snapshot construction/persistence for every Order LineItem in the same slice where Order is born.
- Idempotent `CheckoutCompletionLog` claim/complete/fail behavior.
- Narrow helper for linking `PaymentAttempt.order_id` after Order creation.
- Unit/HTTP tests for replay, concurrency, statuses, correlation and required snapshots.
